# RSSandTwitter2Telegram
RSSとTwitterの更新内容をTelegramに送信するBOTです。GAS(Google Apps Script)で動作します。  
This program is a BOT to send RSS and Twitter updates to Telegram, and it works with GAS (Google Apps Script).
